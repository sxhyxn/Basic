표 만들기

html에서 표를 구현할 때 사용하는 태그

  • tr(table row):행
  • td(table data):열(행의 내용)
  • th(table heading):행,열의 제목(기본값으로 굵은글씨,가운데정렬)
  • 속성 colspan="2(원하는 숫자)" : 2칸의 열병합
  • 속성 rowspan="2(원하는 숫자)" : 2칸의 행병합
